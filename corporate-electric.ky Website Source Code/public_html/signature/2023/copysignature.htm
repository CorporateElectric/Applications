For Gmail
01. Open Gmail.
02. In the top right, click Settings > then "See all settings".
03. In the "Signature" section, add your signature text in the box. If you want, you can format your message by adding an image or changing the text style.
04. At the bottom of the page, click "Save Changes".
-------------------------------- Or --------------------------------
For Outlook
01. On the Home tab, select New Email
02. Select the Message tab. then select Signature, and then choose Signatures/Edit Signatures
03. Under Select signature to edit, Select New and type a name for the signature.
04. Under Edit signature, type the signature that you want to use and select OK.
